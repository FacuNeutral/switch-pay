# Switch Pay
Switch Pay es una solución bancaria diseñada para empresas, que permite a los empleados elegir específicamente qué beneficios desean recibir, según su situación personal y necesidades del momento.
En lugar de recibir beneficios predeterminados, cada persona puede adaptar su experiencia laboral eligiendo entre opciones como viáticos, acceso al gimnasio, almuerzos en la empresa, bonos para compras y más.
