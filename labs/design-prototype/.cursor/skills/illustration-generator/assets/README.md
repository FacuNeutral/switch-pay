# Style References

Las imagenes en esta carpeta son **referencias de estilo** para la skill `illustration-generator`.

## Importante

- **No son ejemplos exactos.** Solo son **acercamientos al estilo de dibujo**.
- **Son ejemplos incompletos.** Les faltan ajustes visuales: proporciones definitivas, sombreado, contraste, detalle de iconos, paleta exacta y composicion final.
- **No usar como ground truth** para la generacion. Tomar la **direccion estetica** (personajes flat, fondos pale-mint, sweaters aqua, ojos ovaloides, hojas decorativas) y **completar** con las reglas explicitas de `SKILL.md` y `reference.md`.

- `style-reference-02.png` — personaje sentado con telefono, fondo pale-mint con planta.
- `style-reference-03.png` — personaje en sillon aqua, espacio compositivo amplio.
